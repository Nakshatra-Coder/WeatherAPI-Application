# Weather API Project 🌤️

A lightweight Python application that fetches and displays current weather details for major cities using an external Weather API.

## 🚀 Features
* Fetches real-time weather data (Temperature, Humidity, Conditions).
* Built completely with custom modular Python logic.
* Simple, clean user interface.

## 🛠️ Prerequisites
Before running this project, ensure you have Python installed on your machine. You will also need a free API key from your chosen weather provider (e.g., OpenWeatherMap).
